# Glossary — SSM Patch Manager Baselines for RHEL and Oracle Linux

Quick definitions of key terms and concepts referenced in this investigation.

---

## InstalledPendingReboot

An SSM metadata property (not an RPM state) indicating patches are on-disk but not active in memory.

## Advisory Prefixes

Vendor-specific naming (e.g., RHSA/ELSA) used to classify patch types.

## Live Patching

Technology allowing kernel updates without a system reboot (kpatch/Ksplice).
